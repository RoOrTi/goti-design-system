---
description: GTLCYSLC - Guardar Todos Los Cambios Y Sincronizar Los Commits
---

Este workflow automatiza la persistencia total de tu trabajo: guarda archivos, crea un commit descriptivo y sincroniza con el repositorio remoto.

// turbo
1. Ejecutar el comando de sincronización maestra (GTLCYSLC):
   ```powershell
   git add . ; git commit -m "GTLCYSLC: Sincronización completa de cambios" ; git push origin main
   ```
