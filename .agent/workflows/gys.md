---
description: GYS - Guardar Y Sincronizar (Commit & Push)
---

Este workflow automatiza la persistencia total de tu trabajo: guarda archivos, crea un commit descriptivo y sincroniza con el repositorio remoto.

// turbo
1. Ejecutar el comando de sincronización maestra (GYS):
   ```powershell
   git add . ; git commit -m "GYS: Sincronización completa de cambios" ; git push origin main
   ```
